Mandelbrot / Julia
Set Viewer

This is a Raylib viewer to see the Mandelbrot set and also the lesser known Julia set family. Very pretty fractals and stuff. The math is incredibly simple, in essence it just sees if a complex point diverges in a sequence.

Web Player

Play around with the simulation inside the web browser using emscripten.

Note that after zoom exceeds e+15, you start seeing precision issues because the regular viewer uses long double to hold position variables. There is an option to use an arbitrary precision library but it is not available in the website because I dont feel like setting up gmp in emscripten actually it might be possible stay tuned i mgiht add it.

In addition it is really hard to get threads working in web assembly so the web version can't use them. Multithreading significantly speeds up the calculation process.

How to Use

Controls: use W/A/S/D to move the view. Press M to zoom in and N to zoom out. Toggle "Using WASD" to drag with the mouse instead.

Iterations: increase the Iterations value for finer detail. More iterations reveal more fractal structure but cost more time.

Performance: increase the number of threads or use low-detail modes to speed rendering. The web build doesn't support multithreading and may be slower, so go download it from github!

Note: after zoom exceeds ~e+15 you may see precision issues; an arbitrary-precision mode exists in the desktop build only.
